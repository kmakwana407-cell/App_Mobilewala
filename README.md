📱 Mobilewala – E-commerce Android Application

Mobilewala is a user-friendly Android-based e-commerce application developed using Android Studio. The app allows users to browse, compare, and purchase mobile phones easily.

It features an attractive and simple UI, making navigation smooth for users. Each product includes detailed information such as images, price, specifications, and description, helping users make informed decisions.

Users can register and log in to their accounts, add products to the cart, and use the “Buy Now” option for quick purchases. After placing an order, users receive confirmation and also have the option to cancel orders within a limited time.

On the admin side, the app provides functionality to add, update, and delete products, making it easy to manage the application.

🔧 Key Features:
User Registration & Login
Product Listing with Images & Details
Add to Cart & Buy Now
Order Confirmation & Cancellation
Simple & User-Friendly Interface
Admin Panel for Product Management
